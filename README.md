## Installation

1. Clone this repository by running the following command in your terminal:
    ```
    git clone 
   ```
2. Navigate to the project directory:
   ```
    cd tech-blog
   ```
3. Install all dependencies by running:
   ```
    npm install
   ```
4. Create Database 
   ```
    mysql -u root & source db/schema.sql
   ```   
5. Start Server 
   ```
    nodemon server
   ```
